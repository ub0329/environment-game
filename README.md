## 소개
- KBC 공모전 '환경'을 주제로한 게임을 만들기위해 개발
- 내용: 귀여운 친구들이 밀실에 갇히게 되지만 쓰레기 정리, 콘센트 뽑기, 분리수거, 환경 퀴즈 등의 다양한 미션을 클리어해야 열쇠를 얻고 나갈 수 있다.
- 어린이들을 대상으로 만든 환경 교육용 게임

## 조작법
- 이동:awsd
- 점프:spacebar
- 클릭:left mouse
